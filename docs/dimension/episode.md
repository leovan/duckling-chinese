## 示例

### 倒数第一集

文本：`倒数第一集`

解析实体列表：

```json
[
  {
    "dim": "Episode",
    "body": "倒数第一集",
    "value": {
      "v": -1.0,
      "unit": "集",
      "dim": "集"
    },
    "start": 0,
    "end": 5
  }
]
```

`最后一集` 具有同上例相同的 `value` 值。
